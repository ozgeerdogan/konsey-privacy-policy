Sık Sorulan Sorular

API anahtarımı nasıl eklerim?
Ayarlar ekranındaki model kartına dokunarak ilgili AI sağlayıcısının (OpenAI, Gemini, Anthropic vb.) API anahtarını girebilirsiniz. API anahtarınız yalnızca cihazınızda şifreli olarak saklanır.

Konuşma geçmişim kayboldu, ne yapabilirim?
Uygulamaya kayıtlı hesabınızla giriş yaptığınızdan emin olun. Hesap giriş bilgilerinizi hatırlamıyorsanız Giriş Yap ekranındaki "Şifremi Unuttum" seçeneğini kullanın.

Hesabımı nasıl silebilirim?
Uygulama içinde Ayarlar → Hesabı Sil seçeneğini kullanarak tüm verilerinizi kalıcı olarak silebilirsiniz.
